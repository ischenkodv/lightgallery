Demo of the LightGallery script

LightGallery is a JavaScript image viewer that shows images above web page and has ability to show both single images and galleries using slide and fade effects. It is very small (8K - packed version, 9,5 - minified) yet powerfull and does not depend on any other libraries. It is tested in all A-grade browsers: IE 6/7, Firefox 2/3, Safari 3, Opera 9.5+.
